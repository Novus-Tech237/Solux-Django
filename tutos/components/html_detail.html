The HTML Fundamentals course is designed to provide a comprehensive introduction to HTML (Hypertext Markup Language), which is the standard markup language used for creating web pages. This course is suitable for beginners who have little to no prior experience with HTML or web development. Throughout the course, you will learn the basic concepts, syntax, and structure of HTML, enabling you to create well-structured and semantically correct web pages.

What You will Learn?

  • Html Syntax and Structure
  • Text Formating and Document Structure
  • Hyperlinks and Navigation within HTML pages
  • How to use HTML assets like images and media
  • Tables and Form Validations
  • Html Shortcuts (emmets) on vscode and Html Semantics
