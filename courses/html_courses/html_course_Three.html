Html Headings

HTML headings are titles or subtitles taht you want display on a webpage

HTML headings are defined with the tags <h1> to <h6>

<h1> defines the most important heading. <h6> defines the least important heading

<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>

Html Paragraph

The Html <p> element defines a paragraph

A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph

<p>This is a paragraph</p>
<p>This is another paragraph</p>

Notes: Despite the number of lines found in the <p> tag the browser will ignore it unless you resize your browser window the number of lines will change

Html Bolds and Strong

The Html <strong> element defines an emphasized text

When this tag is used, the enclosed text is typically bold or more prominent manner by default in most web browsers, although the actual rendering may vary depending on the browser and the specific CSS styles or attributes applied to it

<strong>This is a strong text</strong>

On the other hand, <b> element defines bold formating

The bold tag is a presentational tag that solely focuses on visual appearance of the text, without conveying any specific semantic meaning.

When this tag is used, the enclosed text is displaed in bold by default in most web browsers

<b>This is a bold text</b>

Note: In a nutshell, the bold tag is a presentational tag that is used for bold formating and has any semantic meaning meanwhile the strong tag is used to indicate text emphasis and has a semantic meaning.

In terms of best practice we recommend you to use the <strong> tag when you convey importance or emphasis within your content.

Html Breaks and Preformatted Text

The Html <br> element defines a line break

Use the tag if you want a line break (a new line) without starting a new paragraph

Example

<p>This is a <br> paragraph <br> with line breaks</p>

The <br> tag is an empty tag, meaning that it has no end tag.


The Html <pre> element defines Preformatted text.

The text enclosed in that tag element is a fixed-width font (usually Courier), and it preserves both spaces and line breaks meaning that it preseves the number of lines found in the <pre> tag

Example

<pre>
Roses are red
Violets are blue
That is why you have an error
On line 52
</pre>

Html Spans

The HTML <span> element is an inline-level element that used to group and apply styles to a specific section of text or a specific part of an HTML document.

The "span" element is typically used in conjuction with CSS (Cascade Style Sheets) to apply styles such as color, font etc

<p> Hello, This is a <span style="color: red">Span text with a red color

From the piece of code above you will see that the span is an inline-text within the "p" tag which is styled to have a red color
