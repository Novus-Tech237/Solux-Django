Introduction To HTML

The HTML <head> Element

The HTML <head> attribute is a container that can consist of elements such as: <title> ,<link> ,<style>, <meta> and <base>

The head is displayed on the browser as a tab and the only HTML tag that can be seen is the <title>

The other tags like: <meta>, <meta>, <meta> , ...etc are not seen on the browser

HTML metadata (<meta>) is data about the HTML document and can not be displayed. It typically defines the document tite, the character set, styles and other data information

The HTML <title> Element

This HTML element defines the title of the document. The title must be text only, and as said it is shown on the tab of the web browser.

The content of the page title is very important for the SEO (Search Engine Organization) ! and the page title is used by the search engine algorithms to decide the order when listing pages in search results

Therefore, try to make the <title> content as accurate and meaningful as possible!

A sample HTML document:

Example

<!DOCTYPE html>
<html>
<head>
  <title>A Meaningful Page Title </title>
</head>
<body>
Page Content...
</body>
</html>

The HTML <style> Element

This tag element is used to define the style information of the single HTML page

Example

<style>
body {background-color: powderblue;}
h1 {color: red;}
p {color: blue;}
</style>

The HTML charset Attibute

The character set (charset) is an attribute specified in the <meta>

<meta charset = "UTF-8">

The HTML5 specification encourages web developers to use the UTF-8 character set since UTF-8 covers almost all of the characters and symbols in the world!

The HTML Elements and HTML Attributes

An HTML element is defined by a start tag, some content, and an end tag as seen on Introduction to Basic HTML Tags in Previous Topic

The HTML element is everything from the start to the end of a tag

<tagname>content_of_tag...</tagname>

Examples of some HTML elements:

<h1>My First Heading...</h1>
<p>My First Paragraph...</p>

Demystifying the HTML elements we see according to the example that:

Start Tag Content End Tag
<h1> My First Heading... </h1>
<p> My First Paragraph... </p>

Some HTML element have no content like <br>. These elements are called empty elements and they have no end tag! View All

On the other hand an Attribute is an additional information about HTML elements.

All HTML elements can have attributes and are always specified in the start tag.

Attributes usually come in name/value pairs like: name="value"

Below we will see some important attributes commonly used:

The href Attribute

The <a> tag defines an hyperlink. The href attribute specifies the URL of the page the link goes to:

The Syntax is as follows:

Syntax

<a href ="url">link text</a>

The most important attribute of the <a> element is the href attribute, which indicates the link's destination

By default, the value of the href is an empty string "" and therefore it will not redirect to any link

On clicking on the link text, It will redirect you to the specified url address or html page

Example: Redirecting to URL


This example shows you how to link to Novus Technologies Website
<a href ="novus-tech237.github.io/Novus-Main">Visit Novus Technologies Website</a>

Suppose you have created an html sample page and you want to link to it;

Example: Redirecting to Sample HTML Page


This example shows you how to link to your sample HTML page
<a href ="sample_page.html">Visit My Sample Page</a>

NB: Links can be styled to get another look. By default, Unvisited link is underlined blue, Visited link is underlined purple and Active link is underlined red

The target Attribute

By default, the linked page will be displayed in the current browser window. To change this, you must specify another target for the link.

The target attribute specifies where to open the linked document and it has the following values

  • _self: Default. Opens the document in the same window/tab as it was clicked
  • _blank: Opens the document on a new window/tab
  • _parent: Opens the document in theh parent frame
  • _top: Opens the document in the full body of the window

Example


This Example shows how you open Novus Technologies Website in another tab
<a href ="novus-tech237.github.io/Novus-Main">Visit Novus Technologies Website</a>

The title Attribute

The title specifies extra information about an element. The information is most often shown as a tooltip text when the mouse cursor is over the element.

Example


This Example shows how you open Novus Technologies Website in another tab
<a href ="novus-tech237.github.io/Novus-Main" title ="Go to Novus website">Visit Novus Technologies Website</a>
