Introduction To HTML

What is HTML

Html Stands for Hypertext Markup language used to create a web page

It consists of series of elements called tags which tells the browser how to display the content on the browser.

The actual version of html we will be using will be HTML5 and the code editor will be VsCode (If you haven't started checkout the VSCode Course)

Description of HTML code on Browser

Below is a sample code of a basic html page

Example

< !Doctype html >
< html >
< head >
< title > Page Title < /title >
< /head >
< body >
< h1 > My First Heading < /h1 >
< p > My First Paragraph < /p >
< /body >
< /html >

The above code will then be run on a browser and this is what it will display

Introduction to Basic HTML Tags

A tag is an element attribute that which tells the browser how will the content be displayed on a browser

A tag is written in the form <tagname> ... </tagname>

From the above code we have some various tags that were used in order to run the page on the browser we will see the use of those various tags as seen on the table below

Tag Attribute Function
<!Doctype html> It helps the browser understand how to interpret and render the HTML code that follows
<html> It defines the starting point of an HTML document and provides a container for all other elements.
<head> It contains the information that is found on the browser's tab
<title> This tag labels the name of the browser's tab
<body> This is where all the content of the web page will appear
<h1> This is a heading font
<p> This is a paragraph tag
